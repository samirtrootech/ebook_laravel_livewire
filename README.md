## Installation




1. Run Composer Install
    ```
    composer install
    ```
2. Run NPM Install
    ```
    npm install
    ```
3. Copy .env.example to .env
    ```
    cp .env.example .env
    ```
4. Run NPM Dev
    ```
    npm run dev
    ```
5. Run Migration and Seeding
    ```
    php artisan migrate:fresh --seed
    ```
6. Run the server (Optional)
    ```
    php artisan serve
    ```
7.  Open the browser and go to http://localhost:8000
